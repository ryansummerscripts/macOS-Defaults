<p align="center">
  Automatically set your default preferences for fresh macOS installs!
</p>

<p align="center">
  Dynamically sets compatible preferences based on macOS version, Architecture (ARM vs Intel) and Machine Type (physical vs virtual)
</p>

  Disclaimer: These are my personal preferences.
